# E‑Commerce Website Design — Internship Studio
**Intern:** Yogitha Gattumedhi  
**Duration:** May–June 2025

## Overview
Responsive e‑commerce website implementing browsing, filtering, product view, cart, checkout, order confirmation, about, contact (with validation), and search.

## Features
- Home: navbar, slider, categories, latest products, features, footer, scroll‑to‑top.
- Products: listing with category + price filters.
- Product View: title, images, description, price, ratings, related products, add‑to‑cart.
- Category: category list with short descriptions.
- Cart: quantity, price, subtotal, total.
- Checkout: order summary + shipping form with validation.
- Order Confirmation: message, order number, status.
- About Us: features, testimonials.
- Contact Us: info + form validation.
- Search: product search results.

## Tech Stack
HTML5, CSS3, JavaScript (localStorage), optional images.

## How to Run
Open `index.html` in a browser and navigate via the navbar. Ensure `js/script.js` and `css/style.css` are linked.
